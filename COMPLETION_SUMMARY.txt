DÉMARRAGE RAPIDE
================================================================================

WINDOWS:
  1. Exécuter: setup_ia.bat
  2. python manage.py runserver
  3. Visiter: http://localhost:8000/api/ia/chat/

LINUX/MAC:
  1. Exécuter: bash setup_ia.sh
  2. python manage.py runserver
  3. Visiter: http://localhost:8000/api/ia/chat/

MANUEL:
  1. pip install -r requirements.txt
  2. python manage.py migrate
  3. python manage.py runserver
